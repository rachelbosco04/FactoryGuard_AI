# FactoryGuard AI — Week 2 Completion Report

Project: IoT Predictive Maintenance Engine

Week 2 Focus:
Modeling + Baseline Comparison + Production Model + Hyperparameter Tuning

---

1. Objective

Develop machine learning models to predict rare machine failures
using engineered IoT sensor features from Week 1.

Failure events are highly imbalanced (~0.87% of training data),
therefore precision, recall, F1-score and ROC-AUC were prioritized
over raw accuracy.

---

2. Models Implemented

Baseline Models:

* Logistic Regression
* Random Forest Classifier

Production Model:

* LightGBM Gradient Boosting Model

Hyperparameter Tuning:

* Optuna Bayesian Optimization (50 Trials)
