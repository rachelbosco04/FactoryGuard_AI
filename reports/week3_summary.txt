# FactoryGuard AI — Week 3 Completion Report

Project: IoT Predictive Maintenance Engine

Week 3 Focus:
Imbalance Handling + Explainability + Model Trust

---

1. Objective

Evaluate advanced strategies for rare machine failure prediction
and improve model interpretability for real-world deployment.

Failure events are highly imbalanced, therefore Precision-Recall
metrics were prioritized over raw accuracy.

---

2. Imbalance Handling Strategies Tested

A. LightGBM with Class Weights

* Used scale_pos_weight to prioritize rare failure cases
* Preserved original industrial data distribution
* Low memory overhead
* Faster training

B. LightGBM with SMOTE Oversampling

* Applied synthetic minority oversampling on training data
* Increased failure samples to improve class representation
